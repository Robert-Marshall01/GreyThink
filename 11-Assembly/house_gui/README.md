Before Running
sudo apt install nasm binutils build-essential libx11-dev

To Run
sudo apt install nasm gcc libsdl2-dev
nasm -f elf64 house_gui.asm -o house_gui.o
gcc -no-pie house_gui.o -o house_gui -lSDL2
./house_gui

- Open `house_gui.asm` in your preferred assembler or editor.
