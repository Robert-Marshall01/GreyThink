# Contributor Covenant Code of Conduct

Be respectful and constructive. If you plan to accept external contributions, consider adopting a full Contributor Covenant (https://www.contributor-covenant.org/). This project expects contributors to:

- Be kind and professional in interactions.
- Report security issues privately to the project owner if applicable.
- Treat everyone with respect regardless of background.

If you experience or witness unacceptable behavior, please open an issue or contact the repository owner privately.
