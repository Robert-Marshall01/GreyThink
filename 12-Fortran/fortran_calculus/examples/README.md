# Examples

This folder contains small examples and instructions for running focused subsets of the project for teaching and quick verification.

See the top-level `fortran_calculus.f90` for available demo entrypoints. Example usage:

Unix:

```sh
make build
./fortran_calculus.exe
```

Windows (PowerShell):

```powershell
.\build.ps1
.\fortran_calculus.exe
```
