# Code of Conduct

This project follows the Contributor Covenant 2.1. Please be respectful and professional when interacting with maintainers and other contributors.

Key points

- Be welcoming and inclusive.
- Use respectful language; personal attacks or harassment will not be tolerated.
- If you witness or experience unacceptable behavior, please open an issue or contact the repository owner.

For the full text and reporting guidelines, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
