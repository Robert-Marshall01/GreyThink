# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2024-01-01

### Added

- Initial release
- gRPC service stubs for Auth, User, Projects, Query, and Mutation services
- Domain clients with validation
- Result monad for error handling
- Error codes and normalized GreyError type
- Configuration options with builders
- Complete test suite
